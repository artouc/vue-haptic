# v2.0.0

## Breaking Changes

- React から Vue 3 へ完全移行
  - `useHaptic` が React Hook から Vue Composable に変更
  - `peerDependencies` が `react` から `vue ^3.0.0` に変更
  - 内部実装を Vue の `ref`, `onMounted`, `onUnmounted` ベースに書き換え

## Changes

- 型定義を `src/types/` ディレクトリに分離
- Biome & ESLint Vue を導入
- 命名規則を CLAUDE.md に準拠するよう修正
